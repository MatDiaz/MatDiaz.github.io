Práctica 3

A continuanción se encuentra la práctica 3, correspondiente a Reconocimiento de Filtros. El objetivo será reconocer el filtro aplicado en cada ejercicio. Para esto tendrás que presionar el botón reproducir para escuchar el audio, luego presiona el botón que se encuentra debajo para escuchar las diferencias el filtro que se aplica. Original reproducirá el audio sin cambios y Modificado reproducirá el audio con el filtro respectivo.

A cada nivel de dificultad se agregarán mas tipos de filtros a reconocer y además tendrás un tiempo limite para responder cada ejercicio, si se termina este tiempo, se contará como un error y se generará una incógnita nueva.

Niveles de Dificultad

Fácil: - Pasa Bajas, Pasa Altas. 20 segundos por ejercicio.
Intermedio: - Pasa Bajas, Pasa Altas, Pasa Bandas, Rechaza Bandas, Peaking. 15 segundos por ejercicio.
Avanzado: - Pasa Bajas, Pasa Altas, Pasa Bandas, Rechaza Bandas, Peaking, Dip, Low-shelf, High-shelf. 10 segundos por ejercicio.
