Práctica 2

A continuanción se encuentra la práctica 2, correspondiente a diferencias de nivel. El objetivo será reconocer el cambio de nivel aplicado a cada ejercicio. Para esto tendrás que presionar el botón reproducir para escuchar el audio, luego presiona el botón que se encuentra debajo para escuchar las diferencias de nivel. Original reproducirá el audio sin cambios y Modificado reproducirá el audio con el cambio de nivel respectivo.

A cada nivel de dificultad se le irán sumando mas cambios de nivel a reconocer y además tendrás un tiempo limite para responder cada ejercicio, si se termina este tiempo, se contará como un error y se generará una incógnita nueva.

Niveles de Dificultad

Fácil: -10 dB, 0 dB, + 10 dB. 60 segundos por ejercicio.
Intermedio: -4 dB, -6 dB, -10 dB, 0 db, + 10 dB, + 6dB, +4 dB. 30 segundos por ejercicio.
Avanzado: -1 dB, -2 dB, -4 dB, -6 dB, -10 dB, 0 db, + 10 dB, + 6dB, +4 dB, + 2dB, + 1 dB. 15 segundos por ejercicio.
